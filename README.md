## 💾 Installation

### For Non-Developers (Windows)

1. Go to the [Latest Release](https://github.com/DeepanshuChaid/Cogito-Ai-Memo/releases/latest).
2. Download `cogito.exe`.
3. Move it to a folder and add that folder to your **System PATH** to use it anywhere.

### For Developers (Go installed)

```bash
go install [github.com/DeepanshuChaid/Cogito-Ai-Memo@latest](https://github.com/DeepanshuChaid/Cogito-Ai-Memo@latest)
```
